# IMY 220 Project 2025 – CodeCollab

CodeCollab is a version control and collaboration platform designed for developers to manage, share, and contribute to coding projects. This project is part of the IMY 220 module at the University of Pretoria.
---

## 🚀 Project Purpose

The platform enables users to:

- Collaborate on coding projects
- Track version history with check-in/check-out
- Interact socially through feeds and profiles
- Search and discover projects via tags and users
---

## 🧠 Future Features

- User authentication & role-based access
